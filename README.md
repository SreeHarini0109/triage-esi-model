# Triage Prediction Model

This project predicts Emergency Severity Index (ESI) levels using machine learning.

## Features
- ML model trained on presentation vitals
- FastAPI backend for prediction
- Web UI for input and results

## Tech Stack
- Python
- FastAPI
- CatBoost
- Scikit-learn
- HTML / CSS

## Project Structure
app.py – FastAPI server  
best_esi_model.pkl – trained model  
scaler.pkl – feature scaling  
imputer.pkl – missing value handling  
templates/ – HTML UI  
static/ – CSS/JS  
experiment/ – model training scripts
